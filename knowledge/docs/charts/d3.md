# D3.js

## Selection & Data Binding

```js
import * as d3 from 'd3'

// Select elements
d3.select('#chart')      // single
d3.selectAll('circle')   // multiple

// Data join (modern)
svg.selectAll('circle')
  .data(data)
  .join('circle')
    .attr('cx', d => d.x)
    .attr('cy', d => d.y)
    .attr('r', 5)

// Legacy pattern
const circles = svg.selectAll('circle').data(data)
circles.enter().append('circle')  // new elements
circles.exit().remove()            // removed elements
```

## Scales

```js
// Continuous → Continuous
const xScale = d3.scaleLinear()
  .domain([0, 100])        // data range
  .range([0, 500])         // pixel range

// Discrete → Continuous
const colorScale = d3.scaleOrdinal()
  .domain(['A', 'B', 'C'])
  .range(['red', 'green', 'blue'])

// Discrete → Discrete (bar charts)
const bandScale = d3.scaleBand()
  .domain(['Jan', 'Feb', 'Mar'])
  .range([0, 300])
  .padding(0.1)

// Time scale
const timeScale = d3.scaleTime()
  .domain([new Date(2020, 0, 1), new Date(2020, 11, 31)])
  .range([0, 600])
```

## Axes

```js
const xAxis = d3.axisBottom(xScale).ticks(5)
const yAxis = d3.axisLeft(yScale)

svg.append('g')
  .attr('transform', 'translate(0, 400)')
  .call(xAxis)
```

## Transitions

```js
circles.transition()
  .duration(750)
  .attr('r', 10)
  .style('fill', 'steelblue')
```

## Data Loading

```js
const data = await d3.csv('/data.csv')
const json = await d3.json('/api/data')
```

## SVG 기본 요소

`<rect>`, `<circle>`, `<line>`, `<path>`, `<g>` (group), `<text>`.

**패턴**: select → data → join → attr/style. Scale로 data → pixel 변환. Transition으로 애니메이션.
