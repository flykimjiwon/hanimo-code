# JavaScript ES2024+ Features Reference

## structuredClone
```javascript
// Deep clone (Date, Map, Set 포함)
const original = { date: new Date(), nested: { value: 1 } }
const clone = structuredClone(original)
clone.nested.value = 2 // original은 변하지 않음
```

## Array.groupBy / Object.groupBy
```javascript
const items = [
  { type: 'fruit', name: 'apple' },
  { type: 'fruit', name: 'banana' },
  { type: 'vegetable', name: 'carrot' },
]

const grouped = Object.groupBy(items, item => item.type)
// {
//   fruit: [{ type: 'fruit', name: 'apple' }, { type: 'fruit', name: 'banana' }],
//   vegetable: [{ type: 'vegetable', name: 'carrot' }]
// }
```

## Promise.withResolvers
```javascript
const { promise, resolve, reject } = Promise.withResolvers()

// 외부에서 resolve/reject 가능
setTimeout(() => resolve("done"), 1000)

await promise // "done"
```

## Array.fromAsync
```javascript
async function* asyncGenerator() {
  yield 1
  yield 2
  yield 3
}

const array = await Array.fromAsync(asyncGenerator())
// [1, 2, 3]
```

## String.isWellFormed
```javascript
const valid = "hello".isWellFormed() // true
const invalid = "\uD800".isWellFormed() // false (lone surrogate)
```

## RegExp v flag
```javascript
// Unicode property escapes, set operations
const regex = /[\p{Script=Greek}&&\p{Letter}]/v
regex.test("α") // true
```

## Temporal API (Stage 3 - 곧 출시)
```javascript
// 날짜/시간 처리 (Date 대체)
import { Temporal } from '@js-temporal/polyfill'

const now = Temporal.Now.plainDateISO()
const date = Temporal.PlainDate.from('2024-01-15')
const later = date.add({ days: 7 })

const duration = Temporal.Duration.from({ hours: 2, minutes: 30 })
```

## Decorators (Stage 3)
```javascript
// 클래스/메서드 메타데이터
function log(target, context) {
  return function(...args) {
    console.log(`Calling ${context.name}`)
    return target.apply(this, args)
  }
}

class Service {
  @log
  getData() {
    return "data"
  }
}
```

## Array.at / String.at
```javascript
const arr = [1, 2, 3]
arr.at(-1) // 3 (마지막 요소)

const str = "hello"
str.at(-1) // "o"
```

## Object.hasOwn
```javascript
const obj = { name: "John" }
Object.hasOwn(obj, "name") // true (hasOwnProperty보다 안전)
```

## Error.cause
```javascript
try {
  throw new Error("Failed")
} catch (err) {
  throw new Error("Wrapped error", { cause: err })
}
```

## Top-level await
```javascript
// 모듈 최상위에서 await 사용
const data = await fetch('/api/data').then(r => r.json())
export default data
```

## 자주 쓰는 최신 패턴

### Nullish Coalescing + Optional Chaining
```javascript
const name = user?.profile?.name ?? "Guest"
```

### Private Fields
```javascript
class Counter {
  #count = 0

  increment() {
    this.#count++
  }
}
```

### Logical Assignment
```javascript
obj.value ??= 10 // obj.value가 nullish면 10 할당
obj.value ||= 10 // obj.value가 falsy면 10 할당
obj.value &&= 10 // obj.value가 truthy면 10 할당
```
