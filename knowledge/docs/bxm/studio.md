# 시작하기

BXM Studio를 설치, 업데이트, 로그인 하는 방법에 대해 설명합니다.

## 1. 대상

본 문서는 다음 관련자를 대상으로 합니다.

업무 개발자

프레임워크 관리자

## 2. 실행 환경

BXM Studio의 원활한 실행을 위한 권장 사양은 다음과 같습니다.

| 항목 
| 권장 사양 

메모리

8GB 이상

자바 버전

JDK 11 이상

저장 장치

SSD(Solid State Drive) 10G 이상(사용자 환경에 따라 용량은 증가할 수 있습니다.)

운영체제

Windows 10 64 bit 이상

## 3. 설치

제공되는 압축파일(zip)을 원하는 위치에 복사하고 압축을 해제합니다.

## 4. 실행

압축이 해제된 경로의 BXM5.exe 파일을 더블 클릭하여 실행합니다.

## 5. 로그인

BXM Studio를 사용하기 위해서는 로그인이 필요합니다.

로그인 대화창에서 원하는 서버를 선택하고 계정 정보를 입력 후 로그인 버튼을 클릭하십시오.

## 6. 로그인 서버 지정

로그인에 사용할 서버를 등록할 수 있습니다.

로그인 대화창의 로그인 설정 버튼을 클릭합니다.

로그인 서버 대화창에 필요한 정보를 입력하고 추가 버튼을 클릭하여 서버를 등록합니다.

ID: 서버의 ID를 입력합니다.

이름: 서버의 이름을 입력합니다.

URL: 서버의 주소를 입력합니다.

도메인: 우측의 도메인 조회 버튼을 클릭해 도메인 목록을 조회하고 사용할 도메인을 선택합니다.

작업 공간: 작업 공간을 선택합니다.

기존에 등록된 서버의 수정이 필요한 경우에는 수정할 서버를 선택하고 필요한 정보를 입력합니다. 수정을 클릭하면 입력한 정보가 반영됩니다.

## 7. 작업 공간

정상적으로 로그인이 되면 아래와 같이 BXM Studio의 작업 공간이 실행됩니다.

## 8. BXM Studio 업데이트

새 버전의 스튜디오가 존재하는 경우에 업데이트 메뉴를 통해 업데이트할 수 있습니다.

### 8.1. Update Site 확인

메뉴 &#8594; Help &#8594; 'Install New Software&#8230;&#8203;' 를 선택합니다.

Install 대화창에서 Manage 버튼을 클릭합니다.

Available Software Sites 페이지에서 등록된 Update Site 를 확인합니다.

(1) 등록된 Update Site 를 선택하고 Edit 버튼을 클릭합니다.
(2) Edit Site 대화창에서 Location 필드의 URL을 확인하고 수정이 필요한 경우 URL을 수정합니다.
(3) Add 버튼을 클릭하여 적용합니다.

### 8.2. 수정

메뉴 &#8594; Help &#8594; 'Check for Updates' 를 선택합니다.

새 버전의 BXM Studio가 존재하는 경우에는 다음과 같이 업데이트 목록을 확인할 수 있습니다.

대화창의 Next 버튼을 차례로 클릭하여 업데이트를 완료합니다.

업데이트 완료 후 아래 대화창이 열리면 Restart Now 를 클릭하여 Studio 를 재시작합니다.
