LLM 기초 개념

1. 무엇인가?

Large Language Model(대규모 언어 모델)은 텍스트를 생성/이해하는 인공지능입니다. 수십억 개의 단어를 학습해 문맥을 이해하고, 다양한 문장 생성이 가능합니다.

2. 핵심 구성 요소

  • 텐서: 데이터의 다차원 배열
  • 신경망: 데이터를 처리하는 계산 그래프
  • Transformer: LLM의 핵심 구조(질의응답 기반)

3. 학습 과정

  1. 고해상도 텍스트 데이터 수집
  2. 모델의 가중치 초기화
  3. 손실 함수 기반 최적화 알고리즘 실행
  4. 수천만 회 반복 학습

4. 주요 응용 분야

• 챗봇 개발
• 번역 시스템
• 문서 요약
• 코드 생성
• 질문 응답 시스템

5. 주요 도전 과제

  • 데이터 편향 문제
  • 에너지 소비량 문제
  • 의도적 콘텐츠 생성 시의 윤리적 문제
